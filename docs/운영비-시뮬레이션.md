## 비용 절감 팁

1. **이미지 최적화**: 마이룸 가구 PNG → WebP, 평균 70% 감축
2. **DB 인덱스**: 게시판 카테고리 + created_at 복합 인덱스
3. **Realtime 채널 분리**: 채팅방별 채널, 게시판은 polling 5~10초
4. **Edge Function**: 무거운 집계는 cron으로 야간 배치 → materialized view
5. **CDN**: Storage 직접 노출 X, Cloudflare 무료 캐시 경유
6. **A/B 무료 도구**: PostHog Feature Flag로 외부 도구 절감
