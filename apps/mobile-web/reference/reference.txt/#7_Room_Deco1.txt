#7_Room_Deco1

마이룸 꾸미기 화면을 구현합니다. 화면의 좌측 상단에는 상단 여백 83px, 좌측 여백 20px을 두고 뒤로가기 아이콘을 배치합니다. 뒤로가기 아이콘 아래 20px 위치에는 진행 상태를 나타내는 바를 배치합니다. 이 바는 좌측 여백 20px 기준으로 width 360px, 배경색 #E2E2E2이며, 그 위에 진행도를 표시하는 width 120px의 그라데이션 바(#542BB4 → #E95AA4)가 겹쳐집니다.

진행 바로부터 40px 아래에는 안내 텍스트를 배치합니다. 좌측 여백은 20px이며, 타이틀은 Paperlogy 6 SemiBold 20px로 “거의 다 왔습니다! {닉네임}님의 방을 꾸며보세요!”라고 표시합니다. 이때 {닉네임} 부분은 사용자가 입력한 닉네임이 들어가며 색상은 #7448DD로 강조합니다. 지금은 "무지는 단무지"라는 닉네임을 사용합니다. 타이틀 아래 15px 간격으로 서브 텍스트를 배치하며, Paperlogy 4 Regular 15px, 색상 #A8A8A8로 “취향에 맞는 가구를 골라 나만의 방을 완성해보세요.”라는 문구를 표시합니다.

그 아래 30px 간격으로 메인 미리보기 박스를 배치합니다. 이 박스는 좌측 여백 41px 기준으로 width 320px, height 333px, 배경색 #C4C4C4이며, 추후 이미지가 들어갈 영역이므로 placeholder 형태로 구성합니다.

미리보기 박스 아래 30px에는 선택 가능한 칩 영역을 배치합니다. 좌측 여백은 20px이며 총 12개의 선택 박스가 가로로 나열되고, 한 화면에는 약 5개 반 정도만 보이며 나머지는 가로 스크롤로 확인할 수 있도록 합니다. 각 선택 박스는 텍스트를 감싸는 형태로, padding은 상하 8px, 좌우 18px이며 텍스트 길이에 따라 가로 길이가 유동적으로 변합니다. 기본 상태에서는 배경 없이 border 1px solid #D9D9D9를 가지며, 내부 텍스트는 Paperlogy 4 Regular 16px, 색상 #191919입니다. 선택된 상태에서는 border가 2px solid #7448DD로 변경되고, 텍스트는 Paperlogy 5 Medium 16px, 색상 #7448DD로 변경됩니다. 선택은 하나만 가능하며, 다른 박스를 선택할 경우 기존 선택은 해제됩니다.

그 아래 26px 간격으로 이미지 선택 박스 2개를 배치합니다. 좌측 여백은 20px이며, 두 박스는 가로로 정렬되고 박스 간 간격은 12px입니다. 각 박스의 크기는 width 175px, height 175px이며 배경색은 #F5F5F5입니다. 박스를 선택하면 border 2px solid #7448DD가 적용됩니다.

각 박스 내부에는 이미지가 들어갑니다. 첫 번째 박스의 이미지는 width 123px, height 131px이며, 두 번째 박스의 이미지는 width 108px, height 123px이고, 두 이미지 모두 placeholder 색상 #878787로 표시합니다. 추후 실제 이미지로 교체될 수 있도록 구조를 분리합니다. 사용자가 박스를 선택하면 해당 이미지가 상단 의 미리보기 박스 위에 absolute로 배치되어 반영됩니다.

화면 하단에는 다음 버튼을 고정 배치합니다. 버튼은 좌우 여백 20px 기준으로 width 362px, height 60px, border-radius 30px이며, 텍스트는 “다음”, color #FFFFFF, font-size 18px, SemiBold로 중앙 정렬합니다. 기본적으로 이미지가 2개 이상 선택되지 않은 경우 버튼은 비활성화 상태이며 배경색은 #A8A8A8입니다. 이미지가 2개 이상 선택되면 버튼이 활성화되며 배경색은 #1A1A1A로 변경됩니다. 텍스트는 여전히 #FFFFFF입니다.

이미지가 2개 이상 선택되지 않은 상태에서 다음 버튼을 클릭할 경우 에러 상태를 표시합니다. 버튼 위 8px 위치에 “튜토리얼을 진행해주세요!”라는 문구를 표시하며, 텍스트 색상은 #FF4343, 크기는 14px입니다. 문구 오른쪽에는 동일한 색상의 느낌표 아이콘을 10px 간격으로 배치합니다.
