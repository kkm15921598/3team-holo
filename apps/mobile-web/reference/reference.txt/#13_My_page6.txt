#13_My_page6

마이 탭 안의 내가 쓴 글 화면을 구현합니다. 전체 배경은 흰색입니다.
상단에는 공통 헤더가 있습니다. 화면 상단에서 약 77px 아래 위치에 배치되며, 왼쪽에는 HOLO 로고 텍스트가 있고 오른쪽에는 검색, 알림, 마법봉 아이콘이 가로로 정렬됩니다. 아이콘 간 간격은 16px입니다. 알림 아이콘에는 작은 빨간 점 표시가 있습니다.
헤더 아래에는 페이지 타이틀 영역이 있습니다. 왼쪽에는 뒤로가기 화살표 아이콘이 있고, 그 오른쪽에 내가 쓴 글 텍스트가 배치됩니다. 내가 쓴 글 텍스트는 font-size 20px, font-weight 600으로 표시합니다. 타이틀 영역 하단에는 #D9D9D9 색상의 1px 구분선을 넣습니다.
그 아래에는 목록 상단 제어 영역이 있습니다. 기본 상태에서는 왼쪽에 내가 쓴 글, 오른쪽에 관리하기 텍스트 버튼이 양끝 정렬로 배치됩니다. 이 영역은 padding 20px 정도를 사용하며, 텍스트는 font-size 16px, font-weight 400으로 표시합니다. 관리하기 버튼을 누르면 게시글 삭제 모드로 전환됩니다.
본문에는 내가 작성한 게시글 카드 리스트를 세로로 배치합니다. 리스트 영역은 좌우 20px 여백을 가지며, 각 카드 사이 간격은 24px입니다.
각 게시글 카드는 동일한 디자인으로 반복됩니다. 카드 스타일은 다음과 같습니다.
border-radius: 20px
background: #FFFFFF
box-shadow: 0px 0px 8.3px 0px #DED9F8
padding: 17px
기본 border 없음
카드 내부는 왼쪽 카테고리 라벨과 오른쪽 게시글 정보 영역으로 구성됩니다.
왼쪽 카테고리 라벨에는 운동, 식사, 소분, 공구 같은 카테고리명이 표시됩니다. 카테고리 텍스트는 font-size 16px, font-weight 600 정도로 표시합니다.
오른쪽 게시글 정보 영역은 두 줄로 구성됩니다.
첫 번째 줄에는 왼쪽에 게시글 제목, 오른쪽에 거리와 시간이 표시됩니다.
예시:
러닝 메이트 모집 / 200m · 60분
점심 번개 / 500m · 20분
수박 소분 / 350m · 15분
코스트코 소분 / 700m · 50분
게시글 제목은 font-size 16px, color #000000으로 표시합니다.
거리/시간 텍스트는 font-size 12px, font-weight 500, color #857894로 표시합니다.
두 번째 줄에는 왼쪽에 게시글 설명, 오른쪽에 이동 버튼이 표시됩니다.
예시 설명:
다같이 러닝해요
오피스 단지 떡볶이 메이트!
마트에서 산 큰 수박 나눌 분~
코스트코에서 함께 장 보고 나눌 분~
설명 텍스트는 font-size 14px, color #979797로 표시합니다.
오른쪽 이동 버튼은 width 22px, height 22px, background #CCBCE0, border-radius 20px의 원형 버튼이며, 안에는 대각선 화살표 아이콘이 들어갑니다.
관리하기 모드
기본 상태에서 관리하기 버튼을 누르면 삭제 모드로 전환됩니다.
삭제 모드에서는 상단 제어 영역이 다음과 같이 변경됩니다.
왼쪽 텍스트: 삭제할 게시글을 선택해 주세요
오른쪽 버튼: 삭제하기
삭제하기 버튼 스타일은 다음과 같습니다.
padding: 8px 21px
background: #7448DD
color: #FFFFFF
border-radius: 20px
font-size: 14px
font-weight: 500
삭제 모드에서는 게시글 카드를 선택할 수 있습니다.
선택된 카드는 기존 카드 스타일을 유지하면서 1px solid #7448DD border가 추가됩니다.
여러 개의 카드를 동시에 선택할 수 있습니다.
선택한 상태에서 삭제하기 버튼을 누르면 선택된 게시글 카드가 목록에서 삭제됩니다.
삭제가 완료되면 다시 기본 상태로 돌아가며, 상단 제어 영역은 내가 쓴 글 / 관리하기 상태로 변경됩니다.
선택된 게시글이 없는 상태에서 삭제하기 버튼을 누르면 아무 동작도 하지 않거나 삭제 버튼을 비활성화합니다.
하단 네비게이션 바
화면 하단에는 고정형 bottom navigation bar가 배치됩니다. 탭은 홈, 지도, 게시판, 채팅, 마이로 구성됩니다. 각 아이콘의 크기는 28px × 28px입니다.
선택되지 않은 탭 아이콘 색상은 #A8A8A8, 선택된 탭 아이콘 색상은 #7448DD입니다. 현재 화면에서는 마이 탭이 선택된 상태입니다.
