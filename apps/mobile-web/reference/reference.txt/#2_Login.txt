#2_Login

회원가입 또는 로그인 이전에 보이는 로그인 화면을 구현합니다. 전체 배경은 흰색이며, 상단에는 iOS 스타일 상태바가 있습니다.
상단에서 약 188px 아래에는 HOLO 로고가 배치됩니다. 로고 심볼의 크기는 132px × 112px입니다. 로고 바로 아래에는 HOLO라는 영문 텍스트가 중앙 정렬로 배치됩니다. 텍스트 폰트는 Fredoka 계열이며, 색상은 #B77CFF입니다.
로고 그룹 아래 50px 지점에는 첫 번째 입력창인 이메일 입력 박스가 배치됩니다. 입력 박스 크기는 width 362px, height 62px이며, border 1px, border color #A8A8A8, corner radius 15px입니다. 박스 내부 왼쪽에서 30px 떨어진 위치에 메일 아이콘이 있고, 아이콘 크기는 약 20px × 16px, 색상은 #A8A8A8입니다. 아이콘 오른쪽 20px 지점에는 이메일 아이디 입력 플레이스홀더 텍스트가 있으며, 폰트는 Paperlogy 4 Regular 18px, 색상은 #A8A8A8입니다.
이메일 입력창이 선택되면 해당 입력창의 테두리는 #7448DD 색상의 2px 테두리로 변경됩니다. 이 상태에서는 하단에 iOS 스타일 한글 키보드가 나타납니다. 이메일이 입력된 상태에서는 플레이스홀더 대신 test@naver.com 텍스트가 표시되며, 텍스트 색상은 #1A1A1A입니다.
이메일 입력창 아래 10px 지점에는 두 번째 입력창인 비밀번호 입력 박스가 배치됩니다. 비밀번호 입력 박스 역시 width 362px, height 62px, border 1px, border color #A8A8A8, corner radius 15px입니다. 박스 내부 왼쪽에서 30px 떨어진 위치에 자물쇠 아이콘이 있고, 아이콘 크기는 약 20px × 16px, 색상은 #A8A8A8입니다. 아이콘 오른쪽 22px 지점에는 비밀번호 입력 플레이스홀더 텍스트가 있으며, 폰트는 Paperlogy 4 Regular 18px, 색상은 #A8A8A8입니다.
비밀번호 입력창이 선택되면 해당 입력창의 테두리는 #7448DD 색상의 2px 테두리로 변경됩니다. 테스트용 로그인 계정은 이메일 test@naver.com, 비밀번호 test12345로 설정합니다. 이 정보가 정확히 입력되면 로그인됩니다.
비밀번호를 잘못 입력한 경우에는 비밀번호 입력창만 오류 상태로 표시됩니다. 이때 비밀번호 입력창의 테두리는 #FF4343 색상으로 변경됩니다. 입력창 내부 오른쪽에서 22px 떨어진 위치에는 빨간 원형 경고 아이콘이 표시되며, 아이콘 크기는 18px × 18px, 배경색은 #FF4343, 내부 느낌표 색상은 #FFFFFF입니다. 또한 비밀번호 입력창 아래 9px 지점, 왼쪽에서 약 50px 떨어진 위치에는 비밀번호를 다시 확인해 주세요.라는 오류 문구를 표시합니다. 이 문구의 폰트는 Paperlogy 4 Regular 15px이며, 색상은 #FF1010입니다.
비밀번호 입력창 아래 35px 지점에는 로그인 버튼이 배치됩니다. 버튼 크기는 width 362px, height 60px이며, corner radius는 30px입니다. 버튼 배경은 왼쪽에서 오른쪽으로 #542BB4 → #E95AA4 그라데이션입니다. 버튼 중앙에는 로그인 텍스트가 흰색 #FFFFFF로 표시되며, 폰트는 Paperlogy 6 SemiBold 20px입니다.
로그인 버튼 아래 30px 지점에는 또는 텍스트가 중앙 정렬로 배치됩니다. 이 텍스트는 Paperlogy 4 Regular 15px, 색상 #A8A8A8입니다.
또는 텍스트 아래 15px 지점에는 소셜 로그인 아이콘 3개가 가로로 중앙 정렬되어 배치됩니다. 순서는 구글, 카카오톡, 네이버이며, 각 아이콘 사이 간격은 30px입니다.
소셜 로그인 아이콘 아래 55px 지점에는 회원가입, 아이디 찾기, 비밀번호 찾기 텍스트가 가로로 배치됩니다. 각 텍스트는 Paperlogy 4 Regular 15px, 색상 #A8A8A8입니다. 텍스트 사이에는 동일한 색상의 세로 구분선이 들어가며, 회원가입 | 아이디 찾기 | 비밀번호 찾기 형태로 중앙 정렬합니다.
