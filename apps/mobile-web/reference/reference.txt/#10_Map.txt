#10_Map

지도 메인 화면을 구현합니다. 전체 배경은 흰색입니다. 

상단에서 약 77px 아래 위치에는 헤더가 배치됩니다. 헤더의 왼쪽에는 HOLO 로고 텍스트가 있고, 오른쪽에는 3개의 아이콘이 가로로 정렬됩니다. 아이콘은 검색, 알림, 마법봉 아이콘 순서입니다. 로고와 첫 번째 아이콘 사이에는 충분한 여백이 있으며, 아이콘 간 간격은 16px입니다.

헤더 아래 25px 지점에는 지도 영역이 배치됩니다. 지도 영역의 크기는 width 349px, height 382px이며, 모서리는 둥글게 처리됩니다. 지도는 확대/축소 및 이동이 가능한 인터랙티브 지도 뷰로 구성합니다.

지도 위에는 사용자 주변 모임 위치를 시각적으로 표시합니다. 사용자 위치 근처에는 반투명한 보라색 원형 범위 표시가 있으며, 작은 모임은 작은 반투명 보라색 원과 중앙 아이콘으로, 여러 사람이 포함된 모임은 더 큰 반투명 보라색 원과 중앙의 여러 사람 아이콘으로 표현합니다. 정책상 반경 50m 이내의 위치만 표시되는 구조입니다.

지도 아래 9px 지점에는 안내 문구를 배치합니다. 문구는 ※ 정책상 반경 50m 거리 내 위치만 표시됩니다이며, 폰트는 Paperlogy 5 Medium 14px, 색상은 #ACACAC입니다.

안내 문구 아래 16px 지점에는 3개의 필터 탭이 포함된 세그먼트 컨트롤이 배치됩니다. 전체 크기는 width 319px, height 50px, corner radius 30px이며, 배경색은 연한 보라 계열입니다. 탭 항목은 전체, 지금바로, 계속 함께입니다. 기본 텍스트 스타일은 Paperlogy 5 Medium 18px, 색상은 #A289CD입니다.

선택된 탭에는 그라데이션 pill 형태의 선택 배경이 적용됩니다. 선택 배경 색상은 #542BB4 → #E95AA4 그라데이션이며, drop shadow는 y축 2, blur 6.9, shadow color #542BB4 50%입니다. 선택된 탭의 텍스트는 흰색 #FFFFFF, Paperlogy 6 SemiBold 18px로 표시합니다. 현재 화면에서는 전체 탭이 선택된 상태입니다.

세그먼트 컨트롤 아래 12px 지점에는 모임 카드 리스트가 배치됩니다. 카드 리스트는 가로 스크롤이 가능하며, 현재 화면에는 카드 2개가 주로 보이고 다음 카드 일부가 오른쪽에 잘린 상태로 노출됩니다.

각 카드의 크기는 width 169px, height 153px이며, 배경색은 #EBE4F5, corner radius는 10px입니다. 카드 내부에는 제목, 거리 및 시간 정보, 내용 요약, 우상향 화살표 버튼, 참여자 프로필 아바타가 포함됩니다.

카드 제목은 카드 상단에서 15px 아래, 왼쪽에서 14px 떨어진 위치에 배치합니다. 제목 아래 5px 지점에는 거리와 예상 시간이 500m · 20분과 같은 형식으로 표시됩니다. 거리와 시간 텍스트는 사용자의 현재 위치를 기준으로 계산된 정보입니다.

카드 오른쪽 상단에는 연보라색 원형 버튼이 있으며, 크기는 약 32.85px × 32.85px입니다. 원의 색상은 #CCBCE0이며, 내부에는 북동쪽 방향을 가리키는 화살표 아이콘이 배치됩니다.

카드 본문 내용은 거리/시간 정보 아래 10px 지점에 위치합니다. 내용 영역은 최대 width 94px, height 38px 정도로 제한하며, 이 범위를 넘는 텍스트는 말줄임표(…)로 처리합니다.

내용 아래 8px 지점에는 참여자 프로필 아바타가 표시됩니다. 여러 개의 프로필이 있을 경우 원형 프로필 이미지가 약 -6px 정도 겹쳐지도록 배치합니다.

카드 영역 오른쪽 하단에는 플로팅 액션 버튼(FAB)을 배치합니다. 버튼은 원형이며, 보라색에서 핑크색으로 이어지는 그라데이션(#542BB4 → #E95AA4) 배경을 사용합니다. 중앙에는 흰색 + 아이콘이 들어갑니다.

화면 하단에는 고정형 bottom navigation bar가 배치됩니다. 탭은 홈, 지도, 게시판, 채팅, 마이로 구성되며, 각 아이콘의 크기는 28px × 28px입니다. 선택되지 않은 탭 아이콘 색상은 #A8A8A8, 선택된 탭 아이콘 색상은 #7448DD입니다. 현재 화면에서는 지도 탭이 선택된 상태입니다.

각 아이콘 아래 3px 지점에는 탭 텍스트가 표시됩니다. 기본 텍스트 스타일은 Paperlogy 4 Regular 14px, 색상 #A8A8A8입니다. 선택된 탭 텍스트는 Paperlogy 6 SemiBold 14px, 색상 #7448DD로 변경됩니다.
