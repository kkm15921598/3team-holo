#5_Nickname

회원가입 과정에서 닉네임을 설정하는 화면을 구현합니다. 전체 배경은 흰색이며, 상단에는 iOS 스타일 상태바가 있습니다.
화면 상단 왼쪽에는 뒤로가기 화살표 아이콘이 있으며, 위쪽 여백은 약 83px, 왼쪽 여백은 20px입니다.
뒤로가기 아이콘 아래 20px 지점에는 회원가입 진행률을 나타내는 프로그레스 바가 있습니다. 프로그레스 바는 왼쪽 20px 여백 위치에서 시작하며, 전체 바의 크기는 width 360px, height 5px이고 색상은 #E2E2E2입니다. 그 위에는 현재 진행 단계를 나타내는 width 120px의 가로 그라데이션 바가 겹쳐져 있으며, 색상은 왼쪽 #542BB4에서 오른쪽 #E95AA4로 이어집니다.
프로그레스 바 아래 40px 지점에는 제목 텍스트를 배치합니다. 제목은 왼쪽 20px 여백에 맞춰 정렬하며, 내용은
HOLO에서 사용할
당신의 이름을 정해주세요!
 입니다. 폰트는 Paperlogy 6 Semibold, 크기는 20px, 색상은 #1A1A1A입니다.
제목 아래 15px 지점에는 안내 문구를 배치합니다. 안내 문구는
한글 최대 10자 / 공백, 특수문자 불가
 이며, 왼쪽 20px 정렬입니다. 폰트는 Paperlogy 4 Regular, 크기 15px, 색상은 #A8A8A8입니다.
안내 문구 아래 45px 지점에는 닉네임 입력창을 배치합니다. 입력창은 좌우 20px 여백을 두며, 크기는 width 362px, height 62px, border-radius 15px입니다. 기본 상태의 border는 1px, 색상은 #A8A8A8, 배경색은 흰색입니다.
입력창 내부 왼쪽 22px 지점에는 텍스트를 배치합니다. 기본 placeholder 문구는
닉네임을 입력해 주세요
 이며, 폰트는 Paperlogy 4 Regular, 18px, 색상은 #A8A8A8입니다.
닉네임 입력 조건은 다음과 같습니다.
 한글만 입력 가능
 최대 10자까지 가능
 공백 불가
 특수문자 불가
입력창은 상태에 따라 다음과 같이 변합니다.
1. 기본 상태
 입력값이 없는 상태
 border 색상은 #A8A8A8
 placeholder 텍스트가 보임
 오른쪽 아이콘은 없음
 다음 버튼은 비활성 상태
2. 포커스 상태
 입력창이 선택된 상태
 border 색상은 #7448DD
 placeholder 텍스트는 유지됨
 오른쪽 아이콘은 없음
 하단에는 iOS 한글 키보드가 올라온 상태로 표시
 이 상태에서는 화면 하단의 안내 문구와 다음 버튼이 현재 화면에 보이지 않을 수 있음
3. 성공 상태
 닉네임이 입력 조건을 모두 만족한 상태
 입력창 border 색상은 #7448DD
 입력된 텍스트 색상도 #7448DD
 입력창 오른쪽 22px 지점에는 보라색 체크 아이콘이 표시됨
 다음 버튼은 검정색으로 활성화됨
4. 에러 상태
 공백 또는 특수문자가 포함되어 입력 조건을 만족하지 못한 상태
 입력창 border 색상은 #FF4343
 입력창 오른쪽 22px 지점에는 빨간 느낌표 아이콘이 표시됨
 입력창 아래 10px 지점에는 에러 문구
닉네임을 다시 확인해 주세요.
 를 표시함
 에러 문구는 왼쪽 43px 정도의 위치에서 시작하며, 폰트는 Paperlogy 4 Regular, 15px, 색상은 #FF4343
 에러 상태는 사용자가 조건에 맞게 입력값을 수정하기 전까지 유지됨
화면 하단에는 안내 문구와 다음 버튼을 배치합니다. 안내 문구는
나중에 다시 수정할 수 있어요!
 이며, Paperlogy 4 Regular, 15px, 색상 #A8A8A8, 가운데 정렬입니다.
안내 문구 아래 20px 지점에는 다음 버튼이 있습니다. 버튼은 좌우 20px 여백을 두며, 크기는 width 362px, height 60px, border-radius 30px입니다. 버튼 중앙에는
다음
 텍스트가 표시되며, 폰트는 Paperlogy 6 Semibold, 20px, 색상은 #FFFFFF입니다.
다음 버튼 상태는 다음과 같습니다.
 기본 상태 / 에러 상태: 배경색 #A8A8A8, 비활성
 성공 상태: 배경색 #000000, 활성
에러 상태에서는 입력창 아래의 에러 문구만 추가로 표시되며, 하단의 안내 문구와 다음 버튼은 기존 위치를 유지합니다.
