#12_Chat = 채팅방 상세 화면
채팅 탭 안의 채팅방 상세 화면을 구현합니다. 전체 배경은 흰색입니다.
상단에서 약 77px 아래 위치에는 헤더가 배치됩니다. 헤더의 왼쪽에는 HOLO 로고 텍스트가 있고, 오른쪽에는 검색, 알림, 마법봉 아이콘이 가로로 정렬됩니다. 아이콘 간 간격은 16px입니다. 알림 아이콘에는 작은 빨간 점 표시가 함께 보입니다.
헤더 아래 24px 지점에는 메인 채팅 카드 영역이 배치됩니다. 이 영역은 큰 흰색 카드 형태이며, width는 화면 좌우 여백을 제외한 크기이고, 모서리는 border-radius: 24px로 둥글게 처리합니다. 그림자는 0 0 8.3px #DED9F8 정도로 매우 은은하게 적용합니다.
채팅 카드 내부 상단에는 채팅방 정보 영역이 있습니다. 왼쪽에는 뒤로가기 화살표 아이콘, 그 오른쪽에는 채팅방 이름과 서브 텍스트가 세로로 배치됩니다. 예시 텍스트는 다음과 같습니다.
채팅방 이름: 다같이 러닝해요
서브 텍스트: 살랄라움밥바
채팅방 이름은 검정색, 서브 텍스트는 회색 #9A9A9A로 표시합니다. 오른쪽에는 메뉴 버튼(햄버거/리스트 아이콘)이 있습니다.
채팅방 정보 영역 아래에는 #D9D9D9 색상의 1px 가로 구분선이 들어갑니다.
그 아래에는 실제 채팅 메시지 영역이 있습니다. 메시지는 상대 메시지와 내 메시지가 구분되어 표시됩니다.
상대 메시지
왼쪽 정렬
왼쪽에는 원형 프로필 이미지 표시
프로필 아래에는 닉네임 표시
프로필 오른쪽에는 흰색 말풍선 표시
말풍선 안에는 메시지 텍스트가 들어감
말풍선 옆 또는 아래쪽 오른쪽에는 시간 표시 (15:08)

채팅 카드 하단에는 댓글 입력 영역이 있습니다. 입력창은 긴 둥근 사각형 형태이며, 연한 회색 border 1px와 border-radius: 20px 정도를 가집니다. 왼쪽에는 placeholder 텍스트 댓글 작성하기가 들어가고, 오른쪽에는 전송 아이콘이 배치됩니다. 입력창 좌우 내부 여백은 충분히 확보합니다.
화면 하단에는 고정형 bottom navigation bar가 배치됩니다. 탭은 홈, 지도, 게시판, 채팅, 마이로 구성됩니다. 각 아이콘의 크기는 28px × 28px입니다. 선택되지 않은 탭 아이콘 색상은 #A8A8A8, 선택된 탭 아이콘 색상은 #7448DD입니다. 현재 화면에서는 채팅 탭이 선택된 상태입니다.
각 아이콘 아래 3px 지점에는 탭 텍스트가 표시됩니다. 기본 텍스트 스타일은 Paperlogy 4 Regular 14px, 색상 #A8A8A8입니다. 선택된 탭 텍스트는 Paperlogy 6 SemiBold 14px, 색상 #7448DD로 변경됩니다.
